
# Cave Story Profile Manager

A quick, simple, intutive tool to make changes to your Cave Story Save File! 

<img width="997" height="628" alt="image" src="https://github.com/user-attachments/assets/b054d711-4497-403a-aa77-49b9bcd9a284" />


This tool allows you to make on the fly, quick changes to your inventory. Once any changes are made, simply press the Apply Button to update your Doukutsu.exe's save file! Refer to the Help Button in the Application for more information on how to use!




Currently Supports
- 
Weapon Configurations Including:
- Weapon Levels
- Weapon XP
- Weapon Ammo, Max Ammo
- Weapon Ordering
- Currently Held Weapon

Other Inventory Configurations
- HP and Max HP 

In Development
-

Item Configurations

Flag Configurations

Fun Changes like Music, Player Position, and More!
